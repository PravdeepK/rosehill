# Rose Hill Design Build

Website for [Rose Hill Design Build](https://rosehilldesignbuild.com) — a premium design-build firm specializing in luxury retail fit-outs, restaurants, condos, and commercial spaces.

Built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and deployed on [Vercel](https://vercel.com).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
